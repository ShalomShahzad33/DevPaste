# 🚀 DevPaste

A modern full-stack paste sharing application built with **React, Supabase, Zustand, TanStack Query, and TailwindCSS**.

---

## ✨ Features

- 🔐 Authentication (Email + GitHub OAuth)
- 📝 Create, Read, Update, Delete pastes
- 🔒 Row Level Security (RLS) per user
- ⚡ Fast caching with TanStack Query
- 🧠 Global auth state with Zustand
- 🎨 Modern dark UI (Slate + Blue theme)
- 📄 Individual paste pages
- 🧭 Protected routes
- 🚀 Deployed on Vercel
- 💻 Syntax highlighting for code pastes (Prism.js powered, JS Only)

---

## 🧱 Tech Stack

- React + TypeScript
- Supabase (Auth + Database)
- Zustand (State Management)
- TanStack Query (Server State)
- React Router DOM
- TailwindCSS

---

## ⚙️ Installation

```bash
git clone https://github.com/shalomshahzad33/devpaste.git
cd devpaste
npm install
npm run dev
```
